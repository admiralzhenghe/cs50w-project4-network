# Network

Design a Twitter-like social network website for making posts and following users.

## Architecture

**Frontend**

- Languages:
  - HTML
  - CSS
  - JavaScript

- Framework/Libraries:
  - Bootstrap

**Backend**

- Language:
  - Python

- Framework/Libraries:
  - Django

- Database:
  - SQLite
